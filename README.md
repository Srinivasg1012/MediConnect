# Multi-Tenant MediConnect

A Spring Boot multi-tenant provisioning platform.

## Features

- Dynamic tenant database creation
- Dynamic table creation
- Tenant registry
- Validation
- Global exception handling
- Logging with SLF4J
- Swagger/OpenAPI

## Tech Stack

- Java 21
- Spring Boot
- Spring Data JPA
- MySQL
- Swagger/OpenAPI
- Maven
